## Production Guidelines

### Equipment
- Camera: 1080p minimum
- Microphone: Professional USB microphone
- Screen Recording: OBS Studio or Camtasia
- Editing Software: DaVinci Resolve or Adobe Premiere

### Style Guide
- **Resolution**: 1920x1080 (1080p)
- **Frame Rate**: 30fps
- **Format**: MP4 (H.264)
- **Audio**: 44.1kHz, stereo
- **Branding**: Use Marimo ERP logo and colors
- **Intro/Outro**: 5-second branded intro, 3-second outro

### Recording Tips
1. **Prepare**: Test audio/video before recording
2. **Script**: Follow the script but sound natural
3. **Pace**: Speak clearly and at moderate pace
4. **Errors**: Keep recording, edit mistakes out later
5. **Examples**: Use real, working code examples
6. **B-Roll**: Record extra footage for transitions

### Publishing
- **YouTube**: Marimo ERP official channel
- **Vimeo**: For docs.marimo-erp.com embedding
- **Captions**: Auto-generate and review
- **Chapters**: Add timestamps for sections
- **Description**: Include links to docs and code
